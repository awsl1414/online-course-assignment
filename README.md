# 网络排课系统

## 语言

- 前端 -- vue3
- 后端 -- fastapi

## 是什么

该项目是一个网络排课系统

## 为什么

学校排课异常艰苦，急需一个方便快捷的网络排课系统

## 怎么用

### 1、拉取项目

```
git clone https://github.com/awsl1414/Online-course-scheduling-system.git
```

### 2、启动后端

> 提示：自己安装相关依赖

```
python backend/main.py
```

前端：

> 提示：提前安装 node 环境

```
cd frontend
npm i
npm run dev
```
